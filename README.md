# direct-links-out
Userscript that removes all this "You are leaving our site" and other redirection stuff from links.

__Install:__ https://openuserjs.org/scripts/nokeya/Direct_links_out

Currently supported (with subdomains):  
- facebook.com
- twitter.com
- vk.com
- ok.ru
- deviantart.com
- joyreactor.cc
- steamcommunity.com
